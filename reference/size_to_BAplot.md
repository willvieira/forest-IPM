# Compute Plot-Level Basal Area from Size Distribution

Converts an individual size distribution vector into plot-level basal
area (m2/ha).

## Usage

``` r
size_to_BAplot(N, plot_size)
```

## Arguments

- N:

  List. Size distribution object.

- plot_size:

  Numeric. Plot area in square meters.

## Value

Numeric scalar. Total basal area in m2/ha.
