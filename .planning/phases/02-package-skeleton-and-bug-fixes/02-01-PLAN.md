Replace the entire contents of src/eigen.cpp with the fixed implementation using Eigen::EigenSolver (general, non-symmetric solver):

```cpp
#include <RcppEigen.h>

// [[Rcpp::depends(RcppEigen)]]

using Eigen::Map;
using Eigen::MatrixXd;
using Eigen::VectorXd;
using Eigen::EigenSolver;     // general solver — works on non-symmetric matrices

// [[Rcpp::export]]
VectorXd getEigenValues(Map<MatrixXd> M) {
    // computeEigenvectors=false: faster, only eigenvalues needed
    EigenSolver<MatrixXd> es(M, false);
    // eigenvalues() returns VectorXcd (complex); .real() extracts real parts as VectorXd
    // For IPM kernels (non-negative matrices), dominant eigenvalue is real by Perron-Frobenius
    return es.eigenvalues().real();
}
```

Key changes:
- Remove: `using Eigen::SelfAdjointEigenSolver`
- Add: `using Eigen::EigenSolver`
- Change solver call: `EigenSolver<MatrixXd> es(M, false)` (second arg disables eigenvector computation for speed)
- Change return: `es.eigenvalues().real()` (extracts real parts from VectorXcd to VectorXd)

The function signature `VectorXd getEigenValues(Map<MatrixXd> M)` is unchanged — R-level callers are unaffected.
